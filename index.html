ModalJS library
- By Shlok Jain

This is a javascript library for showing modals in javascript apps, React apps and many more....., It has many features like adding custom buttons, you can choose to make modal draggable or not, This modals are completely editable and you can use HTML to make it's body, and works well in all modern platforms, it is compatible with mobile as well as computer devices. Usage:

Include following CDN links in <head> of your HTML 
                
<head>
    <script src="https://cdn.jsdelivr.net/gh/Shlok-Jain/ModalJS@latest/index.js"></script>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Shlok-Jain/ModalJS@latest/index.css">
</head>
This is a basic example for showing modal 

const modal = new ModalJS({    //initialising modal
    title: 'Title of modal goes here',
    body: 'Your main content of the modal goes here',
    custom_buttons: [
        {
            text: 'SUBMIT',
            theme: 'green',
            onclick: () => { console.log('Thank You for submitting') },
        },
    ],
    close_btn_text: "CLOSE",
    draggable: true,
    close_on_out_click: true,
    onclose:()=>{console.log('closed')}
})

modal.show()       //trigger this when you want to show modal
Explaination:

You are required to pass a JSON as parameter, title contains title of modal and body contains main content,
In the body of modal you can add HTML code also, so you can also create a login or any other custom modal for different purposes.

Now custom_buttons is an array to be passed if you want to show custom buttons.

A default button available on modal will be to close it, You donot need to specify close button in this array, in this array you have to pass another JSON which contains details of button such as text to be represented , theme( green, blue, yellow, red ) , and an onclick listner(an arrow function). You can put any number of buttons just pass them in array in order you want them to be displayed

Other than this some there are some other options available such as
close_btn_text : to specify text written on default close button (put it in capital for fancy look!)
draggable : to make modal draggable on thhe screen
and last but probably not the least
onclose : a function you want to run on closing the modal (optional)

Use the .show() method to show the modal. And if you want to programatically close modal then you can also use .hide() method.

Features:

  • Swipe to close on touch devices
  • You can choose from two themes(light,dark)
  • Pause timer when window loses focus
  • Different types of notifications for different type of messages( normal, alert, error, success )
  • You can manually close notification( .hide() method)
  • hover on notification to pause timer
  • You can specify time to show notification
  • and many more.....
